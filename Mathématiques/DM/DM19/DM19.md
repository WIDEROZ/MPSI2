Comme $P(X_{r} \in X_{r}(\Omega)) = 1$ et par additivité disjointe : 
$$\sum_{i = 1}^{4} P(X_{r} = i) = \frac{1}{2} + 3k = 1$$
Alors, $P(X_{r} = 1) = P(X_{r} = 2) = P(X_{r} = 3) = k = \frac{1}{6}$

On vérifie la fréquence d'apparition des valeurs entre 1 et 4 en répétant un grand nombre de fois $n \in \mathbb{N}$ le lancé de dé.
``` Python 
def valide(Xa, n):
	def frequenceApparition(Xa, n):
		'''Calcul le nombre d'apparition de chaque valeur entre 1 et 4 de la VAR Xa '''
		frequenceApparition = [0, 0, 0, 0]
		while n > 0:
			frequenceApparition[Xa()-1] += 1
			n -= 1
		return frequenceApparition
	
	# Transforme les fréquences d'apparition en probabilité : 
	freqApp = frequenceApparition(Xa, n)
		for i in range(len(freqApp)):
			freqApp[i] = freqApp[i]/n
	
	return freqApp
```

``` Python

```

On a alors : 
$$m_{r+1} = \frac{3(N_{r}+1)+\frac{5}{2}N_{v}+ \frac{7}{2}N_{b}}{N+1}$$
Alors, 
$$m_{r+1} = \frac{3N_{r}+\frac{5}{2}N_{v}+ \frac{7}{2}N_{b}}{N+1} + \frac{3}{N+1}$$
et
$$m = \frac{3N_{r}+\frac{5}{2}N_{v}+ \frac{7}{2}N_{b}}{N}$$
Alors, 
$$m_{r+1} - m = \frac{3}{N+1} + \frac{-\left( 3N_{r}+\frac{5}{2}N_{v} +\frac{7}{2}N_{b}\right)}{N(N+1)}$$
Donc, 
$$m_{r+1}-m = \frac{3}{N(N+1)} \times \left( N-\left( N_{r}+\frac{5}{6}N_{v} +\frac{7}{6}N_{b}\right) \right)$$
Mais encore, 
$$m_{r+1}-m =\frac{3}{N(N+1)} \times \left( N_{v} + N_{b} - \frac{5}{6}N_{v} - \frac{7}{6}N_{b} \right) $$
Donc, 
$$m_{r+1}-m =\frac{1}{2N(N+1)} \times \left( N_{v} - N_{b}\right)$$
Comme $\frac{1}{2N(N+1)}>0$, 
<u>Ainsi, l'espérance dépendra donc du nombre de boules vertes et du nombre de boules bleues</u>
