# Exercice 1
## 1. 
Soit $n \in \mathbb{N}^{*}$, 
On pose $x = \frac{1}{n}$, 
$$\sqrt{ n^{2} + n + 1 } = \sqrt{ \frac{1}{x^{2}}+\frac{1}{x} + 1 }$$
On a : 
$$x \underset{n \to +\infty}{\longrightarrow} 0$$
On fait alors le développement limité de l'expression précédente : 
$$\sqrt{ \frac{1}{x^{2}} + \frac{1}{x} +1 } = 1 + \frac{\frac{1}{x^{2}}+\frac{1}{x}}{2}$$
